❓ Help & Navigation Guide

🏠 Home Page

This is the main page of the Language Learning Center. Use the navigation bar at the top to visit other pages. The cards below show our main courses and what they offer.

📸 Gallery Page

View images of our center and learning activities. Scroll down to see the course schedule table which shows all available courses, levels, timings, and fees. Click the external links at the bottom for additional learning resources online.

📝 Booking Page

Fill out the registration form to book a language course. All fields marked with * are required. Choose your course, level, and preferred days. Write any additional comments before submitting.

  • Name & Email - Your contact information
  • Course Selection - Choose from 5 different courses
  • Level - Select your current language level
  • Preferred Days - Select which days work best for you

👤 About Page

Learn about the creator of this website (Saad Said Araye) and the purpose of the Language Learning Center project. See what technologies were used to build this website.

🛠️ General Tips

  • Hover over images, cards, or table rows to see cool visual effects
  • The table shows course details - compare fees and schedules
  • All forms have HTML5 validation - required fields must be filled
  • The website is responsive - works on mobile, tablet, and desktop
